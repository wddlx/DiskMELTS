## Molecule colour convention

| Molecule | Colour |
|---|---|
| `H2O` / `H2O_hot` | blue |
| `H2O_warm` | C9 (orange) |
| `C2H2` | C3 (red) |
| `HCN` | C4 (purple) |
| `CO2` | C2 (green) |
| `13C12CH2` | C5 |
| `13CO2` | C6 |
| `C4H2` | brown |
| `HC3N` | gray |
